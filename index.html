CLI & Bash-Shell on LINUX-UNIX

  • Below are common commands used in Engineering & I.T. Field
  • pwd : Display your current directory path
  • mkdir fileName : Create new directory
  • cd : Changing, navigating, or getting inside a folder
  • cd.. : Moving to the previous folder
  • cd~ : Navigate to the home directory
  • vi fileName.txt : For creating new text file
  • cat : Displaying file content without going inside the file
  • cat fileName1.txt fileName2.txt > newFileName.txt : Joining two text files together into one file
  • mv originalFileName.txt newFileName.txt : Renaming the file or moving contents to new file while deleting old file.
  • ls : Listing all files and directories
  • ls -R : Listing all files in sub-directories
  • ls -a : Listing hidden files
  • rm fileName.extension : Command for deleting files
  • rm -rf : Remove directory with files
  • cp -R folderName ./newFolder : Copy contents of one folder into another folder
  • ps -f : Displays processes that are running
  • kill -9 : terminates a running process
  • Ctrl + Z : terminates a running program
  • tail -10 fileName.log : Using contents of large error log file
  • ssh root@IP-ADDRESS : Remoting an ip address through the terminal
  • bg%JOB-ID# : Start a job in background
  • scp -S : Command for copying from server to local
  • grep wordNeeded fileName.Extension : Command for copying word from file
  • chmod 777 fileName.txt : Command for changing permissions (read/write/executing) file
  • Fork() : Creates new process from existing process
  • chmod u+w : Gives user writing permission
  • mount -o remount,rw : remount file system as read-write in linux
